## About
Simple calculator to calculate your TDEE (total daily energy expenditure). Using React components and Tailwind for styling. 
## Frameworks used
- React
- Redux
- Tailwind
- Material UI
## Setup
- Download files
- `npm install` to install the required libraries
- `npm start` to run the server
- Navigate to http://localhost:3000/ to view
